# Data Types

## Overview
Data types define the kind of data a variable can hold. They determine how data is stored, processed, and interpreted by the program.

## Common Data Types

### Primitive Data Types
- **Integer**: Whole numbers (e.g., 1, -5)
- **Float**: Decimal numbers (e.g., 3.14)
- **Boolean**: True or False
- **Character**: Single character (e.g., 'a')

### Non-Primitive Data Types
- **String**: Sequence of characters
- **Array/List**: Collection of elements
- **Object**: Collection of properties and methods
- **Dictionary/Map**: Key-value pairs

## Example
```python
count = 10          # Integer
price = 9.99        # Float
is_active = True    # Boolean
name = "Bob"        # String
```

## Type Systems
- **Static typing:** Data type is checked at compile time (e.g., Java, C++)
- **Dynamic typing:** Data type is checked at runtime (e.g., Python, JavaScript)

## Why Data Types Matter
- Ensure correct operations
- Improve performance
- Reduce runtime errors
